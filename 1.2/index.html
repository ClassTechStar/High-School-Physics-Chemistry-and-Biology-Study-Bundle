🇬🇧 高中英语知识体系

📝 语法知识

时态与语态

16种时态，重点：一般现在/过去/将来时、现在/过去完成时、现在/过去进行时。被动语态：be+过去分词。

非谓语动词

不定式(to do)、动名词(doing)、分词(doing/done)。作主/宾/定/状/补语。

三大从句

定语从句(that/which/who/whom/whose)、名词性从句(主/宾/表/同位语)、状语从句(9种)。

虚拟语气

条件句虚拟(与现在/过去/将来相反)、名词性从句虚拟(should+原形)、wish/as if等。

倒装与强调

部分倒装(否定前置/only前置/so neither)、完全倒装(地点状语前置)。强调句：It is...that/who...

📖 词汇与短语

高频词汇

新课标3500词中核心高频词约1500个。一词多义：observe(观察→遵守→庆祝)、address(地址→演讲→处理)。

短语动词

take/bring/come/go/put/get/turn+介词/副词：take up(开始)、take over(接管)、take off(起飞/成功)。

词义辨析

alone/lonely, rise/raise/arise, affect/effect/influence 等近义词区分。

构词法

前缀(un-/dis-/re-/pre-/inter-)、后缀(-tion/-ment/-ness/-able/-ful/-less)、合成词、转化词。

📚 阅读理解

主旨大意题

找主题句(首段/末段/每段首句)，概括全文，选最佳标题。防以偏概全。

细节理解题

定位关键词回原文找对应信息，注意同义替换、正话反说。

推理判断题

基于原文合理推断，不加主观假设。提问词：infer, suggest, imply, conclude。

词义猜测题

利用上下文线索(定义/举例/对比/因果)、构词法、常识猜测词义。

✍️ 书面表达

书信作文

邀请/建议/申请/感谢/道歉/投诉信。格式：称呼+正文+结束语+签名。

议论文

三段式：观点引入→论证(2-3论点+论据)→总结升华。连接词：Firstly, Moreover, In conclusion。

读后续写

新高考25分题型。读懂原文+合理续写两段(150词/段)，风格一致、首句衔接自然。

📖 二、现代文阅读知识点
1.1 信息类文本阅读（论述类+实用类）
核心基础知识点
  1. 核心考查能力：信息筛选与整合、逻辑分析、论证分析、观点推断
  2. 文本三要素：论点、论据、论证
论证方法及答题规范
论证方法核心定义标准答题格式
举例论证列举确凿典型事例证明论点举了……的典型事例，具体有力地论证了……的观点，增强了文本的说服力
道理论证用经典言论、科学原理证明论点引用……的名言/公理，深入论证了……的观点，增强了权威性和严谨性
对比论证正反两方面论点/论据做对比将……和……进行正反对比，鲜明突出地论证了……的观点
比喻论证用熟知的事物作比喻证明论点把……比作……，生动形象地论证了……的观点
因果论证分析论点与论据的因果关系通过分析……的因果逻辑，严谨地论证了……的观点
选择题8大高频易错陷阱
偷换概念：篡改原文概念的内涵/外延
以偏概全：将部分判断扩大为整体判断
混淆时态：将原文"未然/已然/或然"混淆
因果混乱：因果倒置、强加因果
曲解文意：故意错误解读原文语句含义
无中生有：选项内容在原文中无对应依据
混淆是非：将原文肯定/否定内容颠倒
扩大/缩小范围：改变原文限定的范围、程度
主观题答题模板
【论证思路/行文脉络题】
①文章首先提出……的中心论点（开篇方式：开门见山/由现象/事例引出）；
②然后从……等方面，运用……的论证方法，分……个层次展开论证；
③接着进一步分析……/补充论证了……；
④最后总结全文，得出……的结论/发出……的号召。
【论证特点题】（6个维度）
①论证结构：采用……的结构，思路清晰，逻辑严密；
②论证方式：采用立论/驳论/立驳结合的论证方式；
③论证方法：综合运用了……等多种论证方法；
④论点提出：开篇……方式提出中心论点；
⑤论据特点：论据典型、丰富、真实，兼具事实论据与道理论据；
⑥语言风格：语言严谨准确/生动形象，富有感染力。
1.2 文学类文本阅读（小说+散文）
小说核心三要素

人物、情节、环境，核心是人物，最终指向主题。所有题型都可从"人物、情节、环境、主题、读者"5个维度作答。

人物形象分析题
【答题格式】
①总括：XX是一个……（核心性格）的人物形象；
②分点分析：每一点遵循"性格特点+文本具体依据"的格式，分3-4点作答；
③总结：该人物形象的意义/对主题的作用。
情节作用分析题（5大维度）
【答题格式】
对人物：塑造了……的人物形象，突出了人物……的性格/心理；
对情节：为下文……情节做铺垫/埋下伏笔，推动情节发展；
对环境：交代/烘托了……的环境；
对主题：揭示/深化/突出了……的主题；
对读者：设置悬念，吸引读者阅读兴趣。
环境描写作用题
【自然环境描写答题格式】
①交代了故事发生的时间、地点、季节、场景，奠定了全文……的感情基调；
②渲染了……的氛围；
③烘托了人物……的心情/形象，暗示了人物的身份、命运；
④推动情节发展，为下文……情节做铺垫；
⑤暗示/深化了……的主题。
散文阅读核心特点

形散神聚："形散"指选材、结构、手法自由；"神聚"指中心、情感、主旨集中统一。

【行文思路题答题格式】
①文章以……为线索，先写了……，再写了……，接着写了……，最后写了……；
②由……到……，由浅入深，层层递进，抒发了作者……的情感，表达了……的主旨。
【情感主旨题答题格式】
文章通过描写……的景物/事件/场景，抒发了作者……的情感，表达了作者……的观点/人生思考。

文言特殊句式（翻译必考点）
翻译核心要求：特殊句式必须调整为现代汉语正常语序，踩分点明确，漏译句式会扣分。
一、判断句

翻译必须译出"是/不是"

  • ……者，……也。（廉颇者，赵之良将也）
  • ……，……也。（和氏璧，天下所共传宝也）
  • 用"乃、则、皆、本、诚"表判断（当立者乃公子扶苏）
  • 否定判断用"非"（六国破灭，非兵不利，战不善）
二、被动句

翻译必须译出"被"

  • 用"于"表被动（而君幸于赵王）
  • 用"见""见……于……"表被动（臣诚恐见欺于王而负赵）
  • 用"为""为……所……"表被动（若属皆且为所虏）
三、倒装句（高频必考）
  • 宾语前置：否定句中代词作宾语前置（古之人不余欺也）；疑问句中疑问代词作宾语前置（大王来何操）
  • 状语后置：青，取之于蓝，而青于蓝
  • 定语后置：马之千里者；蚓无爪牙之利，筋骨之强
四、省略句

翻译必须补出省略的成分

  • 省略主语：（公）度我至军中，公乃入
  • 省略谓语：一鼓作气，再（鼓）而衰，三（鼓）而竭
  • 省略宾语：不如因而厚遇之，使（之）归赵
断句技巧与翻译方法
断句六大原则
  1. 找名词/代词：常作主语或宾语，前后需断开
  2. 找虚词："曰""盖""夫"等常在句首；"也""矣""焉""哉"等常在句末
  3. 找对话标志："曰""云""言"后常断开
  4. 利用对称结构：排比、对偶、反复中间常断开
  5. 依据语法结构：主谓之间、动宾之间常断开
  6. 参考语境意义：根据上下文语义判断
翻译"六字诀"
留：保留人名、地名、官名等专有名词
删：删除无实义的虚词
补：补充省略的主语、宾语、介词等
换：单音节词换成双音节词
调：调整倒装句的语序
贯：直译为主、意译为辅，语句通顺
翻译踩分点：①重点实词（一词多义、词类活用、古今异义）；②重点虚词；③特殊句式；④句意通顺

答题格式规范
【炼字题标准答题格式】
①释义：解释该字在诗句中的含义，点明是否有词类活用、修辞手法；
②描景：把该字放入原句中，描述诗句展现的画面/景象；
③析效果：该字生动形象地写出了……的特点，烘托了……的氛围；
④点情感：抒发了诗人……的情感。
【表达技巧题标准答题格式】
①准确指出诗句/全诗运用了……的表达技巧；
②结合诗句具体分析，该技巧在诗中是如何运用的；
③分析表达效果：生动形象地写出了……的特点，营造了……的氛围；
④点明情感主旨：抒发了诗人……的情感。
【情感主旨题标准答题格式】
①诗歌通过描写……的景物/场景/事件；
②抒发了诗人……的情感（分点作答，结合诗句分析）；
③表达了诗人……的人生思考/观点态度。
【名句赏析万能公式】
手法 + 释义 + 意境 + 情感
运用了……手法，描绘了……画面，营造了……氛围，表达了……情感。

病句辨析与修改
高考病句六大类型（基础必背）
一、语序不当
  • 多层定语语序不当：正确顺序：领属/时间/处所 + 指称/数量 + 动词短语 + 形容词短语 + 名词短语
  • 多层状语语序不当：正确顺序：目的/原因 + 时间 + 处所 + 范围 + 情态 + 对象
二、搭配不当
  • 主谓搭配不当、动宾搭配不当、主宾搭配不当
  • 一面与两面搭配不当：学习成绩的提高，取决于学生自身是否努力
三、成分残缺或赘余
  • 主语残缺：通过这次学习，使我受到了深刻的教育（"通过"和"使"同时使用）
  • 成分赘余：他同老农多次反复地进行实验（"多次"和"反复"重复）
四、结构混乱
  • 句式杂糅：本着……为原则（正确：本着……的原则/以……为原则）
  • 中途易辙：一句话说了一半，更换主语
病句辨析技巧：①先找句子主干（主谓宾）；②再看修饰成分（定状补）；③关注关联词、否定词、两面词、并列短语；④看是否有句式杂糅、逻辑混乱

试卷结构与评分标准

广东新高考I卷作文以思辨类材料作文为主，满分60分，是高考语文的半壁江山。

等级评分维度核心要求
基础等级40分符合题意准确把握材料内涵，立意符合材料要求，不偏题、不跑题
中心突出有明确的中心论点，全文围绕中心论点展开
内容充实论据充分、典型，能有力支撑论点
结构完整结构完整，首尾呼应，过渡自然；语言流畅
发展等级20分深刻透过现象看本质，揭示内在因果关系
丰富论据丰富，材料充足，论证充分
有文采用词贴切，句式灵活，善用修辞手法
有创意见解新颖，材料新鲜，构思精巧
议论文基础结构模板
【并列式结构】（最易上手）
开头：引述材料，提出明确的中心论点（100-150字）
主体段1：分论点1 + 道理论证 + 事实论据 + 分析论证 + 回扣论点（200-250字）
主体段2：分论点2 + 道理论证 + 事实论据 + 分析论证 + 回扣论点
主体段3：分论点3 + 道理论证 + 事实论据 + 分析论证 + 回扣论点
结尾：总结全文，重申中心论点，升华主题（100-150字）
【递进式结构】（思辨性强，易拿高分）
开头：引述材料，提出中心论点
主体段1：是什么——阐释核心概念的内涵，界定论点的范围
主体段2：为什么——分析论点的意义、价值、重要性
主体段3：怎么办——结合现实，提出践行论点的具体方法
主体段4：思辨段——辩证分析，补充论点，避免绝对化
结尾：总结全文，重申论点，升华主题
标准议论段五步法：观点句 + 阐释句 + 材料句 + 分析句 + 结论句
审题立意技巧
思辨类材料作文审题5步法：
①抓核心要素——通读材料，圈出关键词、关键句，明确核心话题
②辨逻辑关系——分析材料中多个概念之间的关系（对立统一、相辅相成、递进等）
③悟命题意图——思考命题人想要考查的价值导向、时代主题
④立中心论点——提炼出明确、鲜明、有思辨性的中心论点
⑤拟分论点——围绕中心论点，拆分出3个逻辑清晰的分论点
【开头3种基础写法】
①材料入题法：简洁引述材料，分析材料，直接提出中心论点
②名言入题法：引用与论点相关的名言警句，引出中心论点
③排比入题法：用排比句式引出话题，提出中心论点，增强气势
【结尾3种基础写法】
①总结回扣式：总结全文分论点，重申中心论点，首尾呼应
②时代升华式：结合时代背景、青年使命，升华主题
③呼吁号召式：发出号召，引发读者共鸣
基础备考核心建议
  1. 积累素材：分主题积累素材（家国情怀、青春奋斗、科技创新、文化传承等），每个主题积累2-3个经典人物事例、3句名言
  2. 坚持练笔：每周写1篇完整作文，每天写1个标准议论段，保持写作手感
  3. 打磨标题：标题要鲜明、对仗、有文采，直接点明中心论点
  4. 卷面工整：字迹清晰，卷面整洁，卷面分直接影响阅卷老师的评分
